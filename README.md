# react-shining-div

## Install

```bash
npm install --save react-shining-div
```

## Usage

```tsx
import React, { Component } from 'react'

import MyComponent from 'react-shining-div'
import 'react-shining-div/dist/index.css'

class Example extends Component {
  render() {
    return <MyComponent />
  }
}
```
